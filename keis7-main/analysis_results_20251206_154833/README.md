# Результаты анализа отказоустойчивости оборудования

## Содержание папки:

1. **processed_data.csv** - полный набор обработанных данных
2. **analysis_report.xlsx** - сводный отчет в Excel с несколькими листами
3. **analysis_summary.txt** - текстовый отчет с выводами и рекомендациями
4. **trend_efficiency_vs_wear.png** - график тренда КПД
5. **failure_distribution.png** - распределение отказов по типам
6. **efficiency_vs_wear_scatter.png** - зависимость КПД от износа

## Краткие результаты:

- Средний КПД системы: 55.8%
- Всего отказов: 4239

## Как использовать результаты:

1. Используйте рекомендации из analysis_summary.txt для улучшения обслуживания
2. Настройте систему мониторинга на основе выявленных пороговых значений
3. Регулярно обновляйте анализ при поступлении новых данных
